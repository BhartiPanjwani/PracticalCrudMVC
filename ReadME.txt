To Run the Migrations 

1.Please change the connection string as per your requirement.
<add name="CrudContextDemo" connectionString="Server=(LocalDB)\MSSQLLocalDB;Initial Catalog=CRUDDemo;Integrated Security=True"
	  providerName="System.Data.SqlClient" />

2.Create database named "CRUDDemo" on your server (in case of it doesn't get created by default!)

3.Run command "Update-Database" in your package manager console.

4.Build and Run the project and you are good to go!
